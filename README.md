# Leonardo Forner | Portfólio

Bem-vindo à segunda versão do meu site portfólio pessoal! 🚀

Este projeto foi construído para apresentar minhas habilidades como desenvolvedor e meu progresso como estudante de Engenharia de Controle e Automação na UFSC.

## Tecnologias Utilizadas
- **HTML5:** Estruturação semântica do conteúdo.
- **CSS3:** Estilização moderna utilizando Flexbox, variáveis CSS, e design responsivo (Mobile First).
- **FontAwesome:** Ícones vetoriais.
- **Google Fonts:** Tipografia customizada para um design mais atrativo.

## Características do Projeto
- Design 100% responsivo para desktop e dispositivos móveis.
- Navegação fluida com Smooth Scrolling.
- Código focado em semântica, SEO e acessibilidade.
- Cores de alto contraste e efeitos de transição (hover).

## Como rodar localmente
1. Clone este repositório: `git clone https://github.com/leoforner/site2.0.git`
2. Abra o arquivo `index.html` diretamente em seu navegador preferido.
